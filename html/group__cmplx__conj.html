Detailed Description

Conjugates the elements of a complex data vector. The pSrc points to the source data and pDst points to the destination data where the result should be written. numSamples specifies the number of complex samples and the data in each array is stored in an interleaved fashion (real, imag, real, imag, ...). Each array has a total of 2*numSamples values. The underlying algorithm is used: 

for (n = 0; n < numSamples; n++) {
    pDst[(2*n)  ] =  pSrc[(2*n)  ];    // real part
    pDst[(2*n)+1] = -pSrc[(2*n)+1];    // imag part
}

There are separate functions for floating point, integer, and fixed point 32- 16- 8-bit data types.
