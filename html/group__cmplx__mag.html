Detailed Description

Computes the magnitude of the elements of a complex data vector. The pSrc points to the source data and pDst points to the where the result should be written. numSamples specifies the number of complex samples in the input array and the data is stored in an interleaved fashion (real, imag, real, imag, ...). The input array has a total of 2*numSamples values; the output array has a total of numSamples values. The underlying algorithm is used: 

for (n = 0; n < numSamples; n++) {
    pDst[n] = sqrt(pSrc[(2*n)+0]^2 + pSrc[(2*n)+1]^2);
}

There are separate functions for floating point, integer, and fixed point 32- 16- 8-bit data types. Please ensure the appropriate range is used for integers and fixed point data types, as these are at risk for overflow if using too large numbers. Larger than normal errors are possible when using fixed point numbers and very small values.
