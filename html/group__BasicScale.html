Detailed Description

This module contains the glue code for vector scale. The kernel codes (kernels) are in the Module vector scale Kernels. Multiply a vector by a scalar value. For floating-point data, the algorithm used is: 

    pDst[n] = pSrc[n] * scale,   0 <= n < blockSize.
