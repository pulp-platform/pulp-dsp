Detailed Description

This module contains the glue code for vector subtraction. The kernel codes (kernels) are in the Module vector subtraction Kernels. Element-by-element subtraction of two vectors. 

    pDst[n] = pSrcA[n] - pSrcB[n],   0 <= n < blockSize.
